## Tecnologias

**Server:** Node, Express

**Data Base:** PostgreSQL

## Requisitos

  ```
  PostgreSQL v16.2
  NodeJs v20.12.2
  ```
    
## Run Locally

Clona el proyecto

```bash
  git clone https://github.com/DMiranda2214/DB_Tools.git
```

Ve al directorio del proyecto

```bash
  cd DB_Tools/dbToolsNodeJs
```

Instala las dependecias

```bash
  npm install
```

Inicia el servidor

```bash
  npm run start
```
